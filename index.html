Team delivering the application: William Cyze

williamdcyze@lewisu.edu

Last updated proposal date: April 9th, 2021

A brief description of the end user perspective

The end user will be able to locate where the International Space Station is by running the program. It will use a turtle-like window to display a map of the world and where the Space Station is currently over.

Prioritized list of desired features
  • A Turtle-like window
  • A full map of the world
  • Implementation of location data from the Space Station
Non-prioitized list of features
  • Ability to pinpoint the user's location
  • Ability to draw a predicted path

Language and platforms used

The program will be written in python and I will use Visual Studio Code to write the program

Technical features from class knowledge

The technical features that will be implemented from the concepts we have learned in class will be, the use of python and the use of the turtle library.

The technical features from beyond class

The technical features that will be implemented from knowledge beyond class will be, the use of Tkinter GUI, the implementation of a picture into the GUI window, the tracking code, the implementation of the data from the Space Station

What was delivered

I delivered a product that was able to locate the user's location, display a map of the world, uses a Turtle window, and fully implements location data from the Space Station.

What would I add next

I would like to determine a way of determining distance from the user then converting that data into real world units. I would also like to implement the future path of the Space Station.
